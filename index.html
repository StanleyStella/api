Products

Get informations products.

Product object

    Attributes

  • sku

    integer

    The product SKU composed with the Item code and the variant code.

  • name

    string

    The name of the product.

  • description

    string

    Description of the product

  • price

    number

    Price of the product. In cents. Based in euro.

  • weight

    number

    Weight of the product, 1 piece. In grams.

  • size

    boolean

    Existing sizes of the product.
    0 do not exist. 1 exist.

  • color

    array

    Existing colors for the product.

  • range

    number

    Range of the product.

  • gender

    number

    Product gender.
    1 is male, 2 is female, 3 is unisex, 4 is teen.

  • composition

    object

    Composition of the product

  • style

    string

    Style of the product (polo, sweatshirt, tee, etc.)

  • fit

    string

    Fit of the product.

  • maintenance_tips

    number

    Id of the maintenance tips.

  • size_guide

    number

    Id of the size guide.

  • season

    string

    ID for a specific season.
    Eg : Autumn / Winter 2015 -> AW2015

  • certification

    array

    Certification for the product, eg. gots

  • media

    array

    Return media for thumbnails & cover.

  • similar_products

    array

    Contain informations for similar products.

User

Informations about a user.

User object

    Attributes

  • id

    string

    ID of the user.

  • first_name

    string

    First name of the user.

  • last_name

    string

    Last name of the user.

  • email

    string

    Email of the user. Eg. johndoe@example.com.

  • date_of_birth

    number

    Date of the user. No space, no dash, only number composed woth day month year.

  • address_line1

    string

    Adress of a user. Compose with street & number.

  • address_line2

    string

    More informations about the user's adress.

  • address_city

    string

    City of the user.

  • address_state

    string

    State of the user. Optional.

  • address_zip

    string

    Zip code of the user.

  • address_country

    string

    Country of the user.

  • address_country_code

    string

    Country code of the user.

  • currency

    string

    Currency of the user.

  • orders

    array

    All orders IDs of the users.

  • delivery_adress

    array

    All deliveries adress of the users.

  • credit_card

    array

    All credit cards of the users. Stripe manage this part for a security reason.

Definition

GET https://api.stanleystella.com/v1/users/user-id

Shop

Get informations about shop.

Shop object

    Attributes

  • id

    string

    ID of the shop. Created with SP + unique serie of number.

  • name

    string

    Name of the Shop.

  • adress

    string

    Street adress of the shop.

  • zip_code

    number

    Zip code of the Shop.

  • city

    string

    City of the shop.

  • country

    string

    Country of the Shop.

  • iso_code

    string

    Iso code of the country (2 letters).

  • phone_numbers

    array

    Array of phone numbers for the shop
